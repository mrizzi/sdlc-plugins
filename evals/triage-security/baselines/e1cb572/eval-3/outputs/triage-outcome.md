# Triage Outcome for TC-8003

## Decision: Close as Duplicate of TC-7999

TC-8003 is a **same-stream duplicate** of TC-7999. Both issues track CVE-2026-31812 (quinn-proto panic on large stream counts) for the same version stream [rhtpa-2.2]. TC-7999 is already In Progress and has the correct Affects Versions assigned.

## Rationale

| Factor | Detail |
|---|---|
| CVE | CVE-2026-31812 |
| Vulnerable library | quinn-proto (versions before 0.11.14) |
| Current issue | TC-8003 (status: New, Affects Versions: RHTPA 2.2.0) |
| Existing sibling | TC-7999 (status: In Progress, Affects Versions: RHTPA 2.2.0, RHTPA 2.2.1) |
| Stream match | Both issues have stream suffix [rhtpa-2.2] (2.2.x stream) |
| Duplicate rule | Step 4.1: "If a same-stream sibling exists and is open or in progress, close the current issue as Duplicate" |

The version impact analysis confirmed that within the 2.2.x stream, versions 2.2.0, 2.2.1, and 2.2.2 are affected (shipping quinn-proto 0.11.9 or 0.11.12, both below the fix threshold of 0.11.14). Versions 2.2.3 and 2.2.4 already ship the fixed version (0.11.14) and are not affected.

TC-7999 already covers this scope with Affects Versions [RHTPA 2.2.0, RHTPA 2.2.1] and is In Progress. There is no value in maintaining TC-8003 as a parallel tracker.
