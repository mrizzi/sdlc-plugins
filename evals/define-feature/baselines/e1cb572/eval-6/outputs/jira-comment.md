# Jira Comment

## API Call

```
addCommentToJiraIssue(
  cloudId="2b9e35e3-6bd3-4cec-b838-f4249ee02432",
  issueIdOrKey="TC-XXX",
  comment=<see comment content below>,
  contentFormat="adf"
)
```
