## Jira Comment

The comment is posted to the newly created Bug issue using `addCommentToJiraIssue` with ADF `contentFormat`.

### Comment Content

Bug report created with the following sections:

**Included sections:**
- Issue Description
- Steps to Reproduce
- Expected Result
- Actual Result
- Attachments

**Skipped sections:**
- Root Cause
- Suggested Fix
